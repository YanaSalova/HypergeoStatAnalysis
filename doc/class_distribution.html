Detailed Description

Models a hypergeometric distribution for a given set of black and white balls.

This class encapsulates the hypergeometric distribution, characterized by 'a' as the number of black balls, 'b' as the number of white balls in the population, and 'removed' as the number of balls drawn from the population. It computes theoretical probabilities for drawing a specific number of black balls in the sample.
